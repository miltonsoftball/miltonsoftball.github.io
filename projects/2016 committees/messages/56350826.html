2016 Rule Changes

Posted by Kevin Olivieri on April 3, 2016

Hello Board,

To summarize, here are the rule changes for the 2016 Spring Season. We also explained the rule changes to the appropriate division/teams, posted the changes on the home page and updated the rules on the website.

RULE CHANGESÂ FOR THE 2016 SPRING SEASON
Milton Girls Softball has implemented the following rule changes for the 2016 Spring Season.

Freshwomen (Grades 1-2)
No rule changes.

Sophomore (Grades 3-4)*
Base runners can leave the base when the pitcher releases the ball. The runner can only advance to the next base on a batted ball or a walk. Base runners leaving a base early shall be ruled out.
If there is an attempt to tag the runner out before she returns to the base, she can be tagged out.

Junior (Grades 5-6)
Stealing third base is allowed once the pitched ball releases the pitcher's hand.
There is no dropped third strike.
Stealing second base is not allowed.
Base runners leaving a base early shall be ruled out.

Senior (Grades 7-8)
No rule changes.

*We will evaluate the role of coaches and pitching during the early part of the season.
